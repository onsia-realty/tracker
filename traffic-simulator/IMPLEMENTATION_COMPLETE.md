## 📁 생성된 파일 목록

### 메인 실행 파일
- `main.py` - 통합 메인 스크립트 (휴대폰 테더링 모드)
- `setup.bat` - Windows 자동 설치 스크립트
- `start_simulator.bat` - 시뮬레이터 실행 배치 파일
- `start_dashboard.bat` - 대시보드 실행 배치 파일

### 핵심 모듈 (modules/)
- `phone_tethering.py` - **핵심 혁신!** ADB를 통한 휴대폰 테더링 및 IP 초기화
- `device_emulator.py` - 디바이스 에뮬레이션 (모바일 60%, PC 40%)
- `behavior_simulator.py` - 자연스러운 행동 시뮬레이션 (마우스, 스크롤, 클릭)
- `referrer_handler.py` - Referrer 처리 (네이버/구글/소셜/직접)
- `session_logger.py` - SQLite 세션 로깅
- `rank_tracker.py` - 검색 순위 자동 추적
- `__init__.py` - 모듈 패키지 설정

### 대시보드 (dashboard/)
- `app.py` - Flask 웹 서버
- `templates/dashboard.html` - 실시간 통계 대시보드 UI

### 문서
- `README.md` - 전체 사용 설명서
- `QUICKSTART.md` - 빠른 시작 가이드
- `requirements.txt` - Python 패키지 의존성

## 🚀 핵심 기능

### 1. 휴대폰 USB 테더링 모드 ⭐
- **비용 절감**: 월 $500-2,000 → $10-30 (데이터 요금만)
- **자동 IP 변경**: 10회 세션마다 비행기 모드로 IP 초기화
- **수동 제어**: 휴대폰 연결 시에만 작동 (24/7 자동 실행 아님)

### 2. 디바이스 에뮬레이션
- **모바일 기기**: Galaxy S23 Ultra, S22, A54, Pixel 7 Pro, Pixel 6, LG V50, iPhone 14 Pro Max, iPhone 13, iPhone 12
- **데스크톱**: Windows (Chrome, Edge), Mac (Safari, Chrome), 다양한 해상도
- **비율**: 모바일 60%, 데스크톱 40%

### 3. 자연스러운 행동 시뮬레이션
- **마우스 움직임**: 100-200회 무작위 움직임
- **스크롤**: 60-100% 깊이, 위아래 무작위 패턴
- **클릭**: 0-3회 무작위 클릭, 좌표 무작위화
- **체류 시간**: 30-120초

### 4. Referrer 다양화
- **네이버 검색** (40%): 실제 검색 후 결과 클릭
- **구글 검색** (30%): 실제 검색 후 결과 클릭
- **소셜 미디어** (20%): Facebook, Instagram, KakaoTalk, Naver Blog/Cafe
- **직접 유입** (10%): 북마크, URL 직접 입력

### 5. 세션 로깅 (SQLite)
- 성공/실패 여부
- 체류 시간
- 디바이스 정보
- Referrer 정보
- 부정클릭 점수
- IP 주소

### 6. 검색 순위 추적
- 네이버/구글 검색 순위 자동 체크
- 순위 변동 히스토리 저장
- 하루 1회 자동 체크

### 7. 실시간 대시보드
- 오늘/주/월 통계
- 성공률, 평균 체류시간, 부정클릭 점수
- 검색 순위 변동 그래프 (Chart.js)
- 최근 세션 로그 테이블
- 30초마다 자동 업데이트

## 📊 작동 방식

```
1. 휴대폰 USB 연결 대기
   ↓
2. 연결 확인 → 현재 IP 기록
   ↓
3. 세션 실행 (10회)
   - 무작위 디바이스 선택
   - Referrer 실행 (네이버/구글/소셜/직접)
   - 자연스러운 행동 (마우스, 스크롤, 클릭)
   - 체류 (30-120초)
   - 세션 로그 저장
   ↓
4. 10회 완료 → 비행기 모드로 IP 초기화
   - 비행기 모드 ON (5초)
   - 비행기 모드 OFF (10초)
   - 새 IP 확인
   ↓
5. 5-10분 대기 후 3번으로 돌아감
   ↓
6. 휴대폰 연결 끊기면 1번으로 돌아감
```

## 🎯 사용 방법

### 1. 설치
```bash
setup.bat
```

### 2. 휴대폰 설정
1. 개발자 옵션 활성화 (빌드번호 7번 탭)
2. USB 디버깅 ON
3. USB 케이블로 PC 연결
4. USB 테더링 ON
5. ADB 연결 확인 (`adb devices`)

### 3. 실행
```bash
# 시뮬레이터 실행
start_simulator.bat

# 대시보드 실행 (별도 터미널)
start_dashboard.bat
```

### 4. 모니터링
브라우저에서 `http://localhost:5000` 접속
