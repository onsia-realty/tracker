## 1. 오토다비드 (Autodabid)

### 회사 정보
- **회사명**: 주식회사 로보사피엔스
- **대표**: 장현재
- **위치**: 대전광역시 서구
- **특징**: 자동입찰 프로그램이 메인, 부정클릭 방지는 부가 서비스

### 주요 기능
| 기능 | 설명 |
|------|------|
| 자동입찰 | 네이버 파워링크, 쇼핑검색, 파워컨텐츠 순위 자동입찰 |
| 부정클릭방지 | 의심유입 색출 & 차단 |
| 스케줄링 | 요일/시간대별 입찰 가능 |
| 보고서 | 자동 보고서 생성 |
| 키워드 조합 | 키워드 확장 지원 |

### 기술 방식
- **웹 기반**: 별도 프로그램 설치 없이 브라우저에서 사용
- **설치 방법**: 로그인만 하면 됨 (가장 간단)
- **지원 플랫폼**: 네이버, 카카오, 구글, 페이스북

### 가격
- **C type**: 월 1,000원 (소상공인용)
- **프리다비드**: 무기한 무료 (자동입찰만)
- **파트너 상품**: 광고 대행 포함
- **무료 테스트**: 7일

### 장점
✅ **설치가 가장 간단** (스크립트 삽입 불필요)
✅ 자동입찰 + 부정클릭 방지 통합
✅ 가격이 저렴 (월 1,000원~)
✅ 멀티쓰레드 동시 실행 지원

### 단점
❌ 부정클릭 방지 기능 상세 정보 부족
❌ 자동입찰이 메인이라 부정클릭 전문성 낮음
❌ 기술 문서/원리 공개 안됨

## 2. 보라웨어 (Boraware)

### 회사 정보
- **특징**: 정부기관 투자기업
- **추천사**: 제일기획, 동아일보 등 업계 전문가 추천
- **고객센터**: 1661-8270

### 주요 기능
| 기능 | 설명 |
|------|------|
| 100% 자동관리 | 단 한 번 설정으로 자동 동작 |
| IP 변경 추적 | IP 바꿔서 접속해도 추적 |
| 자동 차단 | 부정클릭 자동 감지 후 차단 |
| 환불 신청 | 네이버 환불 신청 자동화 |
| 3단계 경고창 | 홈페이지에 경고 팝업 표시 |
| 모바일 앱 | 전용 앱으로 실시간 알림/차단 |

### 기술 방식
- **스크립트 삽입 방식**: 홈페이지 소스에 스크립트 추가 필요
- **예외 처리**: 네이버 카페/블로그/modoo! 등은 네이버 광고만 수집
- **네이버/구글 연동**: 광고 시스템과 실시간 연동

### 가격
- **기본 요금**: 월 12만 PV까지 기본
- **무료 혜택**: 3만원 쿠폰 + 14일 무료체험
- **파트너 전환**: 모든 솔루션 평생 무료

### 장점
✅ **100% 자동관리** (설정 후 손 안대도 됨)
✅ 정부기관 투자 → 신뢰도 높음
✅ IP 변경해도 추적 가능
✅ 네이버/구글 광고 시스템 직접 연동

### 단점
❌ 스크립트 설치 필요 (기술적 허들)
❌ 12만 PV 초과 시 추가 비용
❌ 추적 원리 상세 미공개

## 3. 스마트로그 (Smlog)

### 회사 정보
- **업력**: 13년 부정클릭방지 사업 경력 (가장 오래됨)
- **위치**: 서울 중구
- **고객센터**: 1544-7813

### 주요 기능
| 기능 | 설명 |
|------|------|
| 멀티레이어 추적 | IP + 쿠키 + 단말기 + 통신사 추적 |
| 공유 IP 판별 | 정상 공유IP와 부정클릭 구분 |
| 자동 경고/차단 | 부정 방법 감지 시 자동 대응 |
| 푸시 알림 | 부정클릭 발생 즉시 알림 |
| 환급 서식 생성 | 네이버 무효클릭 환급 신청 자동화 |
| 불법 프로그램 감지 | 봇/매크로 유입 감지 |
| 채팅 상담 | 방문자와 실시간 대화 |

### 기술 방식 (가장 상세하게 공개)
```
추적 메커니즘:
1. IP 추적 (기본)
2. 쿠키 추적 (기본)
3. 단말기 식별 (핵심) - 같은 단말기 클릭 내역 자동 합산
4. 통신사 정보 수집
5. 행동 패턴 분석 - 기존 방문패턴과 비교
6. 스마트폰 기종 추적

차단 원리:
- 스마트로그 ↔ 네이버 광고 실시간 연동
- 차단 IP는 네이버 광고 목록에서 광고 자체가 안 보임
- 구글 캠페인에도 실시간 차단 IP 등록
```

### 설치 방식
- **스크립트 삽입**: 홈페이지 소스에 분석 스크립트 추가
- **설치 대행**: 스마트로그 팀에 요청 가능 (FTP 설치 지원)
- **네이버 스마트스토어**: 설치 가능
- **Chrome 확장앱**: 추가 기능용

### 가격
- **무료 체험**: 14일 (기능 제한 없음, 자동 과금 없음)
- **정확한 월 요금**: 사이트에서 별도 확인 필요

### 장점
✅ **13년 경력** → 가장 검증된 기술
✅ **단말기 추적** → IP/쿠키 변경해도 추적
✅ **공유 IP 구분** → 거짓 양성(오탐) 최소화
✅ **불법 프로그램 감지** → 봇/매크로 탐지
✅ 기술 원리 가장 상세하게 공개
✅ 네이버/구글 광고 실시간 연동

### 단점
❌ 스크립트 설치 필요
❌ 정확한 가격 체계 확인 어려움

## 🔬 핵심 기술 분석: 브라우저 핑거프린팅

3개 서비스 모두 **IP + 쿠키 이상의 추적**을 한다고 주장하는데, 이건 **브라우저 핑거프린팅** 기술임.

### 핑거프린팅 수집 요소

| 요소 | 수집 방법 | 고유성 |
|------|----------|--------|
| Canvas | HTML5 Canvas API로 도형 그리기 → 해시 | 1000개 중 1개만 동일 |
| WebGL | GPU 정보, 렌더러 정보 | 매우 높음 |
| Audio | Web Audio API 수학적 생성값 | 높음 |
| 화면 | 해상도, 색심도, 픽셀비율 | 중간 |
| 폰트 | 설치된 폰트 목록 | 높음 |
| 브라우저 | User-Agent, 언어, 시간대 | 낮음 |
| 하드웨어 | CPU 코어수, 메모리 | 중간 |

### FingerprintJS (참고용 오픈소스)
```javascript
// 실제 구현에서 사용되는 주요 API들
const fingerprint = {
  canvas: await getCanvasFingerprint(),
  webgl: await getWebGLFingerprint(),
  audio: await getAudioFingerprint(),
  fonts: await getFontsFingerprint(),
  screen: { width, height, colorDepth, pixelRatio },
  hardware: { cores: navigator.hardwareConcurrency, memory: navigator.deviceMemory },
  timezone: Intl.DateTimeFormat().resolvedOptions().timeZone,
  // ... 30개 이상 속성 수집
};

// 해시 생성 → 고유 식별자
const hash = sha256(JSON.stringify(fingerprint));
```

### 추적 정확도
- **동일 브라우저**: 90-95% 정확도
- **IP 변경 시**: 여전히 추적 가능
- **쿠키 삭제 시**: 여전히 추적 가능
- **시크릿 모드**: 여전히 추적 가능
- **다른 브라우저**: 새 방문자로 인식 (한계)

## 🎯 커스텀 구현 시 채택할 장점

### 오토다비드에서 가져올 것
1. **웹 기반 아키텍처** → 별도 설치 없이 브라우저에서 동작
2. **로그인 기반 관리** → 간단한 접근

### 보라웨어에서 가져올 것
1. **100% 자동관리** → 한 번 설정하면 끝
2. **3단계 경고창** → 부정클릭자에게 경고 표시
3. **실시간 광고 연동** → 네이버/구글 광고 자동 차단

### 스마트로그에서 가져올 것 (핵심)
1. **멀티레이어 추적** → IP + 쿠키 + 단말기 + 핑거프린트
2. **공유 IP 판별** → 거짓 양성 최소화 (중요!)
3. **행동 패턴 분석** → 체류시간, 스크롤, 클릭 패턴
4. **불법 프로그램 감지** → 봇/매크로 탐지
5. **푸시 알림** → 실시간 대응
6. **환급 서식 자동 생성** → 네이버 환급 신청 간소화

## 🛠️ 최종 커스텀 구현 방향

### 아키텍처 선택
```
스마트로그 방식 (스크립트 삽입) + 보라웨어 자동화 수준
```

### 핵심 구현 요소

#### 1. 추적 (스마트로그 기반)
```javascript
// 4레이어 추적
const tracking = {
  layer1_ip: getClientIP(),
  layer2_cookie: getCookieId(),
  layer3_fingerprint: getBrowserFingerprint(),  // 핵심
  layer4_behavior: {
    dwellTime,
    scrollDepth,
    mouseMovements,
    clickPattern
  }
};
```

#### 2. 탐지 알고리즘
```javascript
// 리스크 스코어링 (스마트로그 + 보라웨어 방식)
const riskScore = calculateRisk({
  sameFingerprint_multipleClicks: 30,
  lowDwellTime_withClick: 20,
  noScroll_withClick: 15,
  foreignIP: 25,
  vpnProxy: 30,
  botSignature: 80,
});

// 공유 IP 판별 (스마트로그 핵심 기술)
if (isSharedIP(ip) && normalBehavior) {
  // 정상 처리 → 거짓 양성 방지
}
```

#### 3. 자동화 (보라웨어 기반)
```javascript
// 100% 자동 관리
if (riskScore >= 50) logAndMonitor();      // 의심
if (riskScore >= 80) autoBlock();          // 자동 차단
if (riskScore >= 100) addToBlacklist();    // 영구 차단

// 3단계 경고창
showWarningPopup(warningLevel);

// 네이버/구글 광고 연동 차단
syncWithNaverAds(blockedIP);
syncWithGoogleAds(blockedIP);
```

#### 4. 대시보드
```
- 실시간 방문자 현황
- 현장별 방문 통계
- 부정클릭 로그 (증거 데이터)
- 네이버 환급 서식 자동 생성
- 푸시 알림 설정
```
